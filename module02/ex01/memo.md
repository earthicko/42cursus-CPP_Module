```
10000 00000 00000 00000 0000. : 2^23

	-> 128비트 필요

.00000 00000 00000 00000 0000 : 2^-1
.0*125 00000 00000 00000 00000 0000 : 2^-126

	-> 149비트 필요
지수: 127 ~ -126

1000 0000 0000 0000 0000 0000

0.100 0000 0000 0000 0000 0000
  | 이 자리는 지수가 0일 시 2^-1, 지수가 -1일 시 2^-2, 지수가 n일 시 2^n-1 상징
  | 서브노멀에선 2^-127 상징, 즉 지수를 -126로 세팅

```
```
만약 서브노멀시 지수가 -6이라면

1111 = 15 -> -7 -> 8
0001 = 1 -> -7 -> -6
0000 = 0 -> 서브노멀 (지수 = -6)

16비트 부동소수점 서브노멀시
seee emmm mmmm mmmm

     2^-7
       |
0 0000 000 0000 0001

     1234 5678 9012
0000.0000 0100 0000 0000
이러한 부동소수점으로 표현
-> 1.0 * 2^-6
= 0 0001 000 0000 0000
서브노멀 아님

이러한 고정소수점 수는
     1234 5678 9012
0000.0000 0010 0000 0000
-> 1.0 * 2^-7
= 0 0000 100 0000 0000

2^-7의 자리부터 맨티사에 차례대로 복사하면 됨
```
